# Fogo Treasury Command Center

CFO-level live tokenomics and treasury dashboard for Fogo Foundation.

## Live data
- Fogo RPC (`https://mainnet.fogo.io`): TPS, tx throughput, epoch metrics, stake, validator concentration, wallet balances.
- CoinGecko (`fogo`): price, market cap, FDV, volume, circulating/total supply.
- DeFiLlama: Fogo TVL.
